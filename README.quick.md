# Golden Home Điện Biên - Landing Page

Modern landing page built with Nuxt 3, TypeScript, and Tailwind CSS.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Generate static site
npm run generate
```

## Tech Stack

- Nuxt 3
- TypeScript
- Tailwind CSS
- @nuxt/image
- Font Awesome

## Features

- ✅ Responsive design
- ✅ SEO optimized
- ✅ Type-safe with TypeScript
- ✅ Modern UI with Tailwind CSS
- ✅ Image optimization
- ✅ Smooth animations
- ✅ Contact form
